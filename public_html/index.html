EnigMaX 1.1

EnigMaX is a client-side, vanilla JavaScript app. Oh, and it's about ciphering messages!
Using it is dead simple: choose an User Code and a Password, type the message and crypt it! You can safely transmit it to whoever knows your User Code and your Password, without worrying for it to be intercepted and read.You can also set a lifespan for your ciphered message: after it is over, EnigMaX will not decrypt it anymore!

EnigMaX è un'app lato client, sviluppata in JavaSCript puro. Ah, e serve per cifrare messaggi.
Usarla è semplice: scegli un Codice Utente ed una Password, digita il messaggio e criptalo! Potrai poi trasmetterlo a chiunque sappia il tuo Codice Utente e la tua Password, senza preoccuparti che venga intercettato e letto. Puoi anche impostare una scadenza per il tuo messaggio cifrato: passata quella, EnigMaX non lo decifrerà più.
